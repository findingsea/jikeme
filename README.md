## 安装

```shell
go get -u github.com/findingsea/jikeme
```

> 前提：安装好 Go 语言环境。

## 功能

在终端执行 `jikeme` 命令，即可出发摸🐟！

目前支持的圈子：

* 浴室沉思
* 无用但有趣的冷知识
* 今日烂梗
* 弱智金句大赏
* 今日金句
* 一个想法不一定对
* 有一点哲学在里面
* 记一件小事
* 科技圈大小事
